Sobre o Projeto

Este projeto foi desenvolvido como parte do Case Prático da AutoU.

Utiliza Inteligência Artificial para classificar e-mails como produtivos ou improdutivos, sugerindo respostas automáticas para otimizar o fluxo de trabalho.

Tecnologias utilizadas:
  • Backend: Python com FastAPI e LlamaIndex.
  • IA: API da Groq com modelo de linguagem.
  • Frontend: HTML, CSS, Javascript, jQuery, Bootstrap 5 e SweetAlert2.
